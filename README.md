# 🏗 스마트코브라 경매 자동 분석 시스템

매일 오전 7시 부산·김해·양산 경매 물건을 자동 검색하고
AI 분석 결과를 카카오톡으로 알림 보내는 시스템입니다.

## 검색 조건
- **농지**: 전·답·임야 / 100~200평 / 7천만원 이하 / 유찰 1회↑
- **주거·상가**: 빌라·단독·상가주택·근린상가 / 2억 이하 / 유찰 1회↑
- **지역**: 부산 전체 + 김해시 + 양산시

## 분석 항목 (100점 만점)
1. 입찰가 추천 (보수적/적정/공격적)
2. 권리분석 (인수권리 여부)
3. 체류형쉼터·스마트팜 가능 여부
4. 예상 수익률
5. 건축적 하자 검토

## 설치 방법

### 1. 카카오톡 연동 (최초 1회)
```bash
pip install -r requirements.txt
python kakao_setup.py
```

### 2. Render.com 환경변수 설정
Render 대시보드 → Environment → 아래 값 추가:
- `KAKAO_REFRESH_TOKEN`: kakao_setup.py 실행 후 발급된 값
- `ANTHROPIC_API_KEY`: Anthropic API 키

### 3. 자동 실행
Render에 배포하면 매일 오전 7시 자동 실행됩니다.
